### ♻️ How to Restore a Deleted User in Office 365

This section outlines how to **restore a user** from the **Deleted Users** tab in the Microsoft 365 Admin Center. This process is helpful when users are accidentally removed or need to be reactivated.

---

#### 🔍 Step 1: Navigate to Deleted Users

On the left navigation pane, click **"Deleted users"**.  
Find and select the user you wish to restore.

![Step 1 – Deleted Users Tab](https://github.com/user-attachments/assets/3452ccf7-c3ff-471d-8033-339f639e7694)

---

#### 🔐 Step 2: Restore User and Set Password

Once you click **Restore**, you'll be prompted to either **assign a password manually** or allow the system to **auto-generate a password**.

![Step 2 – Restore & Password](https://github.com/user-attachments/assets/881a0265-23e3-44b6-af4f-2d5304e90183)

---

#### 📋 Step 3: View Restoration Summary

After completion, a summary screen will appear confirming that the user has been successfully restored.

![Step 3 – Restored User Summary](https://github.com/user-attachments/assets/289d133b-70bc-44a0-8625-2afb6283ecb9)

---

### 📌 Notes:
- Restored users regain their previous username and license assignment (if available).
- If the license pool is full, a warning will be shown during the restore process.
- Users restored within **30 days** retain most of their data and settings.
